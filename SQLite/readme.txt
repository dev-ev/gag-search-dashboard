The script gagFeatureDB_create.py inititates an empty SQlite database.

The script gag_Minora-to-SQlite.py is to be integrated into the Scripting Node
in Proteome Discoverer 2.4 or later.
It reads the output from the Minora LC-MS feature detection and saves the quantitative info
on the chromatographic peaks into the SQLite database.
